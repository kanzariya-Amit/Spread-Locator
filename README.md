# 📊 Spread-Locator  
### Statistical Analysis of Transaction Dataset  

## 📌 Project Overview  
This project focuses on the application of statistical concepts and probability distributions to analyze a real-world transaction dataset. It is structured into two key components: theoretical understanding and practical implementation, ensuring both conceptual clarity and analytical application.

---

## 🧠 Part A: Theoretical Foundation  
Part A establishes a strong foundation in statistical theory by covering essential concepts required for data analysis and distribution modeling.

### Topics Covered:
- Statistical Distributions  
- Q-Q Plot  
- Discrete vs Continuous Distributions  
- Bernoulli Distribution  
- Binomial Distribution  
- Log-Normal Distribution  
- Power Law Distribution  
- Box-Cox Transformation  
- Poisson Distribution  
- Z-score Probability  
- PDF (Probability Density Function)  
- CDF (Cumulative Distribution Function)  

Each concept is explained with clear definitions and concise explanations to build a solid understanding of statistical behavior in datasets.

---

## 💻 Part B: Practical Implementation  
Part B focuses on applying statistical techniques to the dataset using Python to extract meaningful insights and identify underlying patterns.

### Tasks Performed:
- Loading and exploring the dataset  
- Identifying relevant variables for statistical analysis  
- Applying Bernoulli distribution to model binary outcomes  
- Simulating Binomial distribution for repeated trials  
- Modeling event frequency using Poisson distribution  
- Analyzing transaction amounts using Log-Normal distribution  
- Fitting Power Law distribution to examine data patterns  
- Generating Q-Q Plot to assess normality  
- Applying Box-Cox transformation to improve distribution  
- Calculating Z-scores to measure deviation from the mean  
- Visualizing Probability Density Function (PDF)  
- Visualizing Cumulative Distribution Function (CDF)  

---

## 📊 Key Insights  
- Transaction outcomes can be effectively modeled using Bernoulli and Binomial distributions.  
- Event frequency in the dataset follows a pattern that can be approximated using the Poisson distribution.  
- Transaction amounts exhibit positive skewness and are better represented by a Log-Normal distribution.  
- Data transformation techniques such as Box-Cox significantly improve distribution characteristics.  
- Visualization tools like Q-Q plots, PDF, and CDF enhance understanding of data behavior and distribution patterns.
